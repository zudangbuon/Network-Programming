# Network-Programming

\## B1 — Multi-Client File Transfer Server

\*Concurrent upload/download with progress reporting\*



\*\*Description:\*\* Build a central file server that accepts multiple simultaneous client connections.

Clients can list available files, upload files to the server, and download files from it. 

Transfer progress must be shown during every operation. 

Files must be processed in chunks — never loaded entirely into memory.
